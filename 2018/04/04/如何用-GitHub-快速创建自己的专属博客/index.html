如何用 GitHub 快速创建自己的专属博客(一)

04-04-2018
本文总阅读量次

Github 为用户提供了 Pages 的功能, 使得静态网页的展示变得非常的容易.
不仅如此, 每一个Github 的用户都可以创建一个基于github 端的博客. 简单易操作.
阅读本文, 只需要5分钟.

  1. 首先你要用有一个 Github 账户. 注册传送门

  2. 创建一个 Repository. 选择 Repository.

  3. 点击右边 New 按钮.

  4. 设置参数, 这里要注意, 每一个账户只能有一个这个格式的 Repository, 如图所示.
    你的用户名.github.io

  5. 点击 Create Repository 按钮, 你的博客到这一步已经建立成功了. 现在你可以通过 Repository 的名字来访问你的博客啦.

  6. 如果想要自定义博客的域名, 可以通过 setting, 或者创建一个 CNAME 的文件, 里面写上你的自定义域名就可以了.
    这里需要注意的是, github 默认的是 https 的协议, 如果自定义域名没有使用 ssl 的话, 那么域名默认是 http protocal.

    在你的 Repository 页面上方选择 settings.
